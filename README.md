Mon portfolio personnel minimaliste, construit avec HTML et CSS vanilla. Pas de framework, pas de build, juste du code propre.

## Apercu

Un site one-page responsive qui presente mon parcours, mes projets et mes coordonnees. Concu pour etre rapide, accessible et facile a maintenir.

## Tech Stack

- **HTML5** - Structure semantique
- **CSS3** - Flexbox, Grid, animations CSS
- **GitHub Pages** - Hebergement gratuit et fiable
- **Google Fonts** - Typographie (Inter)

## Installation locale

```bash
git clone https://github.com/nemesyts/P1RXJV4OP.git
cd P1RXJV4OP
# Ouvrir index.html dans un navigateur
# ou utiliser un serveur local :
python -m http.server 8000
```

## Deploiement

Le site est automatiquement deploye via GitHub Pages a chaque push sur la branche `main`.

URL : [https://nemesyts.github.io/P1RXJV4OP/](https://nemesyts.github.io/P1RXJV4OP/)

## Personnalisation

1. Modifier le contenu dans `portfolio/index.html`
2. Ajuster les styles dans `portfolio/style.css`
3. Remplacer les images dans `portfolio/assets/`
4. Push sur `main` pour deployer
