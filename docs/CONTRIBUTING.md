# Contributing

Merci de vous interesser a ce projet. Les contributions sont les bienvenues.

## Comment contribuer

1. Forkez le repository
2. Creez une branche pour votre modification (`git checkout -b feature/ma-modification`)
3. Commitez vos changements (`git commit -m 'Ajout de ma modification'`)
4. Pushez sur votre branche (`git push origin feature/ma-modification`)
5. Ouvrez une Pull Request

## Guidelines

- Gardez le code simple et lisible
- Pas de framework JS, on reste en vanilla
- Testez sur mobile avant de soumettre
- Les images doivent etre optimisees (max 200KB)

## Signaler un bug

Ouvrez une issue en decrivant :
- Le comportement attendu
- Le comportement observe
- Votre navigateur et OS
- Des captures d'ecran si possible

## Questions

Pour toute question, ouvrez une issue avec le tag `question`.
